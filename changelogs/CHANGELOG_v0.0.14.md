# Changelog

## [v0.0.14] - 2026-03-04

### Added
* **Stack-based Folding Range Provider** to support folding of **nested constructs**, improving navigation in complex CKSP files.

### Changed
* Updated **syntax highlighting** for the `override` keyword when used in **function definition headers** and **callbacks**.
* Removed the **border around VS Code dropdown elements** in the compile view for a cleaner UI.

### Fixed
* Fixed issue where **downloading a new CKSP compiler version** would not automatically update the **workspace settings** to use the newly installed version.
